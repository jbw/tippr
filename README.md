## Getting Started

Make sure you have installed and configured docker in your environment. After that you can run the below commands from the root of the project. 

```sh
docker-compose build
docker-compose up
```

## Local Development

### Install packages
```sh
npm install
```

### Test
```sh
npm run test
```

```sh
npm run test:e2e
```

### Run locally
```sh
npm run start:dev
```

```sh
http://localhost:3000
```
